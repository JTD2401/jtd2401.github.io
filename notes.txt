Use an id for a specific element
<div id="exampleID">
#exampleID {}
Use classes for similar elements
<div class="exampleClass">
.exampleClass {}

Difference between figure, div, and section:
Figure should be used with a picutre of pictures and <figcaption>
to add captions to it
Div holds stuff
Section defines sections in a document such as chapters, headers, footers, etc.

Change object within class/id using >
Example:
.buttonLink:hover > li {
  background: orange;
}
